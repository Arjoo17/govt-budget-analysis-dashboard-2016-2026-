# Government Budget Analysis Dashboard (2016–2026)

## Project Overview
This project is an **Excel Data Analysis Dashboard** that analyzes government budget allocation and spending trends from **2016 to 2026**. The dashboard provides interactive visualizations to understand how government funds are distributed across sectors and states and how spending trends change over time.

The project demonstrates **data cleaning, pivot table analysis, and interactive dashboard creation in Microsoft Excel**, which are important skills for data analysis roles.

---

## Objectives
- Analyze government budget allocation across different sectors.
- Identify states receiving the highest budget allocation.
- Track government spending trends over multiple years.
- Visualize beneficiaries across different sectors.
- Provide an interactive dashboard using slicers.

---

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Cleaning and Transformation

---

## Dataset
The dataset contains government budget information from **2016 to 2026** with the following fields:

- Year
- State
- Sector
- Budget Allocation
- Beneficiaries

---

## Dashboard Features

### 1. Budget Allocation by Sector
Shows how government funds are distributed across sectors such as:
- Agriculture
- Education
- Employment
- Health
- Infrastructure
- Rural Development
- Urban Development
- Women & Child Welfare

This helps identify which sectors receive the highest funding.

---

### 2. Top States by Budget Allocation
A horizontal bar chart showing the states receiving the highest budget allocation.

Examples:
- Uttar Pradesh
- Tamil Nadu
- Punjab
- Maharashtra
- Kerala

---

### 3. Government Spending Trend (2016–2026)
A line chart showing the change in total government spending over time.

This helps identify whether spending is increasing or decreasing across years.

---

### 4. Beneficiaries by Sector
Displays how many beneficiaries are impacted by different government programs across states and sectors.

---

### 5. Interactive Filters (Slicers)

The dashboard includes interactive slicers to filter the analysis by:

- **Sector**
- **State**
- **Year**

Users can dynamically explore the data by applying these filters.
